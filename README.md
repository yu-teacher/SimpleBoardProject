# 🌟 Simple Board Project 🌟

게시판의 최소한의 기능을 만든 웹사이트입니다. 회원가입, 게시글, 댓글 기능이 있습니다.

---

## 📚 프로젝트 설명

Simple Board Project는 기본적인 게시판 기능을 갖춘 웹 애플리케이션입니다. 사용자들은 회원가입을 통해 로그인한 후, 게시글을 작성하고, 댓글을 달 수 있습니다.

---
